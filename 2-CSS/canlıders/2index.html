CSS Başlangıç

CSS syntax

Selector:

Bir CSS kuralının hangi HTML öğelerine uygulanacağını belirtir.

Property

Bir CSS ozelliginin adidir.

Ozellikle belirli bir ogenin gorunumu veya davranısı konrol den ozelliktir.

Declaration

Bildirim blogu, noktalı virgul ile ayrılmıs bir veya daha fazla bildirim icerir.

Value

Bir ozelligin belirli bir niteligidir. Ozelliklerin degerleri, ozelliklerin belirledigi ozellerin turune baglı olarak degisir.

CSS Selectors

Bir Css secici , bicimlendirmek istediğimiz HTML ogelerini secer.

CSS element Selector

Elemena seçiçi HTML ogelerini etiket adına göre seçer

Exm. Burada, sayfadaki tüm <p> öğeleri kırmızı metin rengiyle ortalanmış olacak:
p {
text-align: center;
color: red;
}

CSS id Selector

Kimlik secici belirlie bir ogeyi secmek için bir html ogesinin id niteligi kullanılır.
